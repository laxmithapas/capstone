# HDFC Bank - Digital Account Opening

A premium, responsive frontend implementation of the HDFC Bank digital account opening flow. This project demonstrates a modern user interface with HDFC branding, featuring a multi-step form, drag-and-drop document upload, and interactive validation.

## Features

- **Premium UI/UX**: Designed with HDFC's corporate identity (Navy Blue & Red) using modern design principles (Glassmorphism, floating labels).
- **Responsive Design**: Fully responsive layout that adapts seamlessly from desktop to mobile devices.
- **Client-Side Validation**: Real-time form validation for inputs like mobile numbers, email, and required fields.
- **Drag & Drop Upload**: Interactive file upload zone supporting drag-and-drop functionality with file type/size validation.
- **Progressive Disclosure**: Clean multi-step form layout to guide the user through the process.
- **Simulation Mode**: Simulates backend processing with a loading state and success modal generation.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Grid, Flexbox, and CSS animations. No external frameworks (Vanilla CSS).
- **JavaScript**: ES6+ for DOM manipulation, event handling, and form logic.
- **FontAwesome**: For UI icons.
- **Google Fonts**: Inter font family for modern typography.

## Project Structure

## How to Run

1.  Clone this repository or download the files.
2.  Open `index.html` in any modern web browser.
3.  No local server is required (works via `file://` protocol), though a Live Server is recommended for the best experience.

## Usage Guide

1.  **Fill Personal Details**: Enter Name, DOB, Mobile, and Email.
2.  **Address Section**: Provide address details.
3.  **Upload Documents**: Drag and drop a dummy image/PDF (max 5MB).
4.  **Submit**: Click "Submit Application" to see the success simulation.
